## 프로젝트 시작 방법!!

1. clone level-1 repository

```
git clone https://github.com/tekit-school-five-member/level-1.git
```

<br>

2. 본인 이름의 branch 생성

```
git checkout -b <yourName>
```

<br>

3. 본인 이름 브랜치의 feature 브런치 생성

```
git checkout -b feature/<yourName>
```
